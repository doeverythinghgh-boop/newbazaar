# نظام إدارة الطلبات

نظام متكامل لإدارة حالة الطلبات، البائعين، وخدمات التوصيل مع دعم كامل للصلاحيات والأدوار المختلفة.

│   │   ├── stepper.css       ← تنسيقات شريط التقدم

## الصفحات

### 1. الصفحة الرئيسية (index.html)
- عرض روابط لجميع الصفحات
- معلومات عن المشروع والأدوار المدعومة

### 2. شريط التقدم (order-view.html)
- عرض جميع حالات الطلب: REVIEW → CONFIRMED → SHIPPED → DELIVERED → CANCELLED → REJECTED → RETURNED
- ملخص الطلب مع إحصائيات البائعين

### 3. تفاصيل البائعين (vendor-details.html)
- عرض بيانات كل بائع
- الخطوات الفرعية لكل بائع
- حالة كل بائع (كامل / جزئي / فشل)

### 4. خدمات التوصيل (delivery-options.html)
- إدارة خدمات التوصيل لكل بائع
- اختيار الخدمة المعتمدة
- عرض خطوات التوصيل

### 5. لوحة الاختبار (test-lab.html)
- مركز التحكم الكامل
- تغيير الأدوار والصلاحيات
- تعديل بيانات البائعين وخدمات التوصيل
- محاكاة حالات مختلفة

## الأدوار والصلاحيات

| الدور | الصلاحيات |
|------|-----------|
| **المشتري** | تعديل الكميات المطلوبة فقط |
| **البائع** | تعديل الكميات المتوفرة، تغيير الحالة من REVIEW → CONFIRMED |
| **شركة الشحن** | إدارة خدمات التوصيل، تغيير حالات SHIPPED و DELIVERED |
| **الإدارة** | جميع الصلاحيات بما فيها إلغاء ورفض وإرجاع الطلبات |

## حالات الطلب

1. **REVIEW** - مراجعة
2. **CONFIRMED** - مؤكد
3. **SHIPPED** - شُحن
4. **DELIVERED** - تم التسليم
5. **CANCELLED** - ملغي
6. **REJECTED** - مرفوض
7. **RETURNED** - مرتجع
